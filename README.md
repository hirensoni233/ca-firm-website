# Hiren K Soni & Co - CA & Accounting Automation Firm

A professional, premium website built for a Chartered Accountant specializing in rigorous financial management, tax advisory, and advanced accounting automation.

## Tech Stack
- **Framework:** React + Vite
- **Routing:** React Router DOM
- **Styling:** Custom CSS + CSS Modules
- **Animations:** Framer Motion
- **Icons:** React Icons

## Features
- **Premium Design:** "Tech-forward" aesthetic with glassmorphism, glowing accents, and animated nodes.
- **Scroll Animations:** Smooth Framer Motion `whileInView` scrolling reveals.
- **Lead Generation:** Integrated active `FormSubmit` contact form.
- **Responsive Layout:** fully optimized for mobile, tablet, and desktop viewing.
- **Active Navigation States:** Visual indicator on current active route.

## Local Development
To run this project locally:

1. Clone the repository
2. Install dependencies:
```bash
npm install
```
3. Start the development server:
```bash
npm run dev
```

## Deployment
This project is fully ready to be deployed to **Netlify** or **Vercel**. 

1. Push this repository to GitHub.
2. Link the repository in your Netlify dashboard.
3. Use the build command: `npm run build`
4. Publish directory: `dist`
